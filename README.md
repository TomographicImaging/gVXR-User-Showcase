# gVXR-User-Showcase
A repository to showcase jupyter-notebooks created by users of gVirtualXray (gVXR). 
These notebooks are contributed by users of gVXR. 
Some have been lightly edited by gVXR maintainers for clarity, but credit for the contents remains with their authors.

For an interactive introduction to gVXR, created by gVXR developers, see our [tutorial notebooks](https://github.com/TomographicImaging/gVXR-Tutorials/).

## Contribution Guidelines

To contribute to the repository,

1. Create a fork of the repository and a new branch.
2. Create a new folder, containing your contributions, in the form of jupyter notebooks, with any necessary supporting python files, and a LICENSE file.
3. Add a description of your contribution(s) to the top of your file(s)
4. If publicly available, add a link to the dataset used near the top of the file(s)
5. Add the gVXR version you ran with near the top of the file(s)
6. Add license headers to all of the files (with a license of your choice)

When your contributions are ready, open a pull request for the gVXR developer team to review! 
When you open the pull request please tick the box saying "Allow edits from maintainers".

The gVXR developer team will then review your contributions and may ask for small changes or push some small changes to your fork.

If you wish to contribute a notebook, have any questions or want to find out more then please join the [gVXR community on Discord](https://discord.gg/VVmcP9Ty).
